# 🛡️ ISMS-P 기반 보안 자동화 프로젝트
## 1. 프로젝트 개요
이번 프로젝트는 **KISA ISMS-P 인증 기준(2.11.2 취약점 점검)**을 바탕으로 진행되었습니다.

## 2. 진단 대상 (Target OS & DB)
총 3개의 OS와 2개의 DB를 대상으로 보안 진단 도구를 구축합니다.
* **Ubuntu** (Master Node)
* **Rocky Linux 9 & 10**
* **MySQL & PostgreSQL**
* **Ubuntu**

### 🔍 주요 기능: 쉘 스크립트 진단
KISA 보안 리스트의 모든 항목을 정밀한 쉘 스크립트를 통해 보안 자동화 진단 및 조치까지 수행합니다.
